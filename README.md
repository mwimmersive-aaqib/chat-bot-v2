# Brazil eVisa Premium Application

This project hosts a multi-step Brazil eVisa application experience built with Next.js and Tailwind CSS. It features inline validation, a progress tracker, document upload guidance, FAQ modal, and a save-and-resume flow via `localStorage`.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the app.

## Building for Production

```bash
npm run build
npm run start
```

## Deployment

The repository includes a `vercel.json` configuration for deployment on Vercel. Push the project to a Git repository and import it into Vercel; the default build command (`npm run build`) and output directory (`.next`) are already configured.
